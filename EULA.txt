5. DATA PRIVACY AND LOCAL OPERATION
------------------------------------
The Software is designed with privacy as a core principle.

  5.1 FILES READ LOCALLY
      The Software reads the following files on your local machine and
      does not upload, sync, or transmit their contents to any external
      server controlled by the Author:

        - ~/.claude/sessions/*.json         (session metadata)
        - ~/.claude/projects/**/*.jsonl     (conversation logs)
        - ~/.claude/.credentials.json       (OAuth token; see 5.3)
        - %APPDATA%\WhereMyTokens\*         (local app data, including
                                             live-session.json written by
                                             the optional statusLine
                                             bridge plugin)

  5.2 FILES MODIFIED LOCALLY (OPT-IN)
      If -- and only if -- you explicitly enable the "Claude Code
      Integration" feature from within the Software's Settings, the
      Software will modify the file ~/.claude/settings.json on your
      machine to register itself as a Claude Code statusLine plugin.
      This modification is reversible from within the Settings screen.
      The Software will not modify this file without your explicit action.

      Additionally, if you enable "Launch at login," the Software will
      register a standard Windows auto-start entry for the current user.

  5.3 ANTHROPIC API REQUESTS
      If Anthropic credentials are present on your system in the location
      referenced above, and you have not disabled rate-limit polling, the
      Software makes periodic HTTPS requests to Anthropic's usage API
      endpoint on your behalf using your own credentials, to retrieve
      your own rate-limit information. These requests are functionally
      equivalent to requests you would make yourself by logging into the
      Anthropic Console.

      You are solely responsible for ensuring that your use of the
      Software in combination with your Anthropic credentials complies
      with Anthropic's applicable terms of service, usage policies, and
      rate limits. The Author is not a party to any agreement between
      you and Anthropic and accepts no responsibility for any
      consequences arising from such use, including but not limited to
      account suspension, throttling, or termination.

  5.4 NO TELEMETRY, NO CLOUD SYNC
      The Software does not collect crash reports, usage statistics,
      analytics, or any other diagnostic data. No account, registration,
      or internet connection is required for the core local functionality
      of the Software.

  5.5 THIRD-PARTY SERVICES
      The Software does not communicate with any third-party services
      other than Anthropic's own API using your own credentials as
      described in Section 5.3.

  5.6 LOCAL SECURITY
      You are solely responsible for the security of your local machine
      and any credentials stored on it.

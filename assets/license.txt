4. DATA PRIVACY AND LOCAL OPERATION
------------------------------------
WhereMyTokens is designed with privacy as a core principle:

  - LOCAL FILES ONLY: The software reads the following files solely on your
    local machine and does not upload, sync, or transmit their contents to
    any external server:
      * ~/.claude/sessions/*.json         (session metadata)
      * ~/.claude/projects/**/*.jsonl     (conversation logs)
      * ~/.claude/.credentials.json      (OAuth token, used only to fetch
                                          your own usage data from Anthropic)
      * %APPDATA%\WhereMyTokens\*        (local app data)

  - NO TELEMETRY: The software does not collect crash reports, usage
    statistics, analytics, or any other diagnostic data.

  - NO CLOUD SYNC: No account, registration, or internet connection is
    required for the core functionality of this software.

  - ANTHROPIC API: If your Anthropic credentials are present, the software
    makes periodic requests to Anthropic's usage API endpoint on your behalf
    (as if you were viewing your own dashboard) to retrieve rate limit
    information. These requests are identical to requests you would make
    yourself by logging into the Anthropic Console.

  - THIRD-PARTY SERVICES: The software does not communicate with any
    third-party services other than Anthropic's own API using your own
    credentials.

You are solely responsible for the security of your local machine and any
credentials stored therein.
